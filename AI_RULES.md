# AI Development Rules

## 项目定位
AI Investment Platform 是一个 AI 驱动的个人投资研究平台。

目标：帮助普通投资者按照机构投资流程进行研究和决策。

## 开发原则

1. 所有功能先设计后开发。
2. 所有需求必须形成文档或 Issue。
3. 代码必须包含测试。
4. 优先模块化设计，方便 Agent 自动维护。
5. 所有重大修改必须经过 Pull Request。

## 技术规范

Backend:
- Spring Boot
- Java
- MySQL
- Redis

Frontend:
- Vue3

AI:
- LLM
- RAG
- Agent

Deployment:
- Docker
- Linux
- CI/CD
