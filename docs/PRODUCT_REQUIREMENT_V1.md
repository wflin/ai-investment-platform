# AI Investment Platform V1 产品需求

## 产品定位

打造 AI 驱动的机构级投研工作台，帮助个人投资者建立系统化投资研究流程。

目标不是预测股票涨跌，而是辅助用户完成市场理解、行业研究、公司分析和投资复盘。

## V1目标

第一阶段服务个人投资者使用，完成行业研究闭环。

每天自动生成 AI 行业晨报：

- 市场环境
- 行业排名
- 行业评分变化
- 热点行业分析
- 风险提示

## 核心研究流程

宏观环境 -> 市场状态 -> 行业分析 -> 板块分析 -> 公司研究 -> 财务分析 -> 估值分析 -> 风险分析 -> 投资复盘

## V1范围

### 行业研究引擎

采用申万行业分类体系。

覆盖：
- 一级行业
- 二级行业
- 股票与行业关联

### 行业评分

评分维度：

- 景气度
- 资金
- 政策
- 估值
- 新闻情绪

输出综合评分和排名。

### AI分析

生成结构化行业报告：

- 行业现状
- 驱动因素
- 风险因素
- 重点关注方向

## 开发原则

- 分阶段开发
- 小步交付
- 每个Sprint产生可运行成果
- AI结论需要可解释
