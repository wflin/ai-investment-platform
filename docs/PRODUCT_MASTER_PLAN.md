# AI Investment Platform 产品总纲

## 1. 产品定位

AI Investment Platform 是 AI 驱动的个人投资研究平台。

目标：帮助普通投资者建立接近机构的投研流程，通过数据、AI Agent 和系统化方法辅助投资研究。

原则：
- 不预测短期涨跌。
- 不提供盲目交易信号。
- 帮助用户理解宏观、政策、行业、公司和风险。

## 2. 核心投研流程

宏观环境 → 政策分析 → 市场周期 → 行业研究 → 板块轮动 → 公司研究 → 估值分析 → 投资组合 → 风险控制 → 复盘优化

## 3. 系统架构

### 数据层

负责管理：
- A股行情数据
- 指数和成交量数据
- 行业分类数据
- 公司财务数据
- 新闻数据
- 政策数据

### AI分析层

通过多个Agent完成研究。

### 应用层

提供：
- 行业分析
- 股票研究
- AI投研报告
- 投资组合管理
- 风险提醒

## 4. AI Agent体系

### 投资总监Agent

负责综合研究、投资逻辑整理和报告生成。

### 宏观Agent

分析经济、利率、汇率、流动性和全球市场。

### 政策Agent

负责政策采集、政策解读、影响行业分析。

流程：
政策 → 行业 → 公司 → 投资机会与风险

### 行业Agent

分析：
- 行业趋势
- 景气度
- 政策支持
- 资金流向
- 估值水平

### 股票Agent

分析：
- 公司基本面
- 财务质量
- 盈利能力
- 竞争优势
- 估值
- 风险

### 风控Agent

负责仓位、回撤和风险管理。

面向更多投资者提供服务。

## 6. 行业研究体系

行业结构：
一级行业 → 二级行业 → 细分赛道 → 股票

行业评分：
- 政策支持
- 资金流向
- 行业景气
- 盈利趋势
- 估值水平
- 技术趋势

## 7. 政策分析体系

建立政策知识库：

政策来源 → 政策内容 → 影响行业 → 影响公司 → 机会与风险

关注来源：
- 国务院
- 发改委
- 工信部
- 央行
- 证监会

## 8. AI开发流程

需求设计 → PRD → GitHub Issue → Codex开发 → Pull Request → CI测试 → 自动部署 → 企业微信通知

## 9. 开发原则

- 所有功能先设计后开发。
- 所有模块必须文档化。
- 所有代码必须测试。
- Agent职责清晰。
- 架构优先考虑长期扩展。
